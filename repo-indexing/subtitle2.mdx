# Subtitle2 Component

## Overview

### Purpose

Use this component for secondary subtitle text styling

### When to Use

The Subtitle2 component is designed for secondary subtitle text that provides tertiary hierarchy below main titles and primary subtitles. Use Subtitle2 when you need:

- Sub-section headings that provide additional organization within content areas already structured by Subtitle1
- Secondary subtitle text that supports primary subtitles with more granular categorization
- Descriptive labels for content groups, card headers, or form sections
- Text that needs more prominence than body text but less than Subtitle1
- Consistent typography for secondary subtitles that maintains visual hierarchy
- Subtitle text in content-heavy interfaces where multiple levels of subtitle hierarchy are needed

### Architecture

**No Coupling** - Subtitle2 is a standalone typography component that doesn't require other components to function. It's part of the broader typography system but operates independently, providing secondary subtitle-level text styling.

### Component Interdependencies

Subtitle2 has no required dependencies and can be used as a standalone component. However, it's part of a cohesive typography hierarchy that includes related components like Title1, Title2, Title3, Subtitle1, Body1, and other text styling variants for different emphasis levels and content purposes.

## Components

### Subtitle2 {Standalone Component}

**Purpose:** Provides secondary subtitle text styling that adheres to the Fluent Design System's typography standards. This component ensures consistent font sizing, line height, and spacing for secondary subtitle content that establishes tertiary hierarchy in your interface.

**Interface:**

```typescript
interface Subtitle2Props {
  /**
   * The content to be rendered as secondary subtitle text
   */
  children?: React.ReactNode;

  /**
   * The HTML element or React component to render
   * @default 'span'
   */
  as?: React.ElementType;

  /**
   * Applies block-level styling (display: block)
   * @default false
   */
  block?: boolean;

  /**
   * Prevents text wrapping (white-space: nowrap)
   * @default false
   */
  nowrap?: boolean;

  /**
   * Applies text truncation with ellipsis on overflow
   * @default false
   */
  truncate?: boolean;

  /**
   * Text alignment
   */
  align?: "start" | "center" | "end" | "justify";

  /**
   * Additional CSS class names
   */
  className?: string;

  /**
   * Inline styles
   */
  style?: React.CSSProperties;
}
```

**Usage Example:**

```tsx
import { Subtitle2 } from "@fluentui/react-components";

function Example() {
  return <Subtitle2>Product Categories</Subtitle2>;
}
```

## Best Practices

### Usage Guidelines

- Use Subtitle2 for tertiary headings that provide granular organization within sections already structured by primary titles and Subtitle1
- Leverage the `as` prop to ensure semantic HTML structure (h3, h4, etc.) for proper accessibility and document outline
- Combine with the `block` prop when you need the subtitle to occupy full width and create visual separation from adjacent content
- Use consistent spacing patterns when combining Subtitle2 with other typography components to maintain visual rhythm

### Accessibility

- Always use appropriate heading levels (h3, h4, h5, etc.) with the `as` prop to maintain logical document structure
- Ensure heading hierarchy is sequential and doesn't skip levels (h1 → h2 → h3, not h1 → h3)
- Use Subtitle2 consistently across your application to maintain predictable navigation patterns for screen readers
- Consider contrast ratios when applying custom styling to ensure readability across different themes

### Performance

- Subtitle2 components are lightweight and optimized for performance as part of the FluentUI design system
- Use semantic HTML elements when possible for better SEO and accessibility
- Avoid excessive nesting or overly complex styling that might impact rendering performance
- Consider using `truncate` prop for long text in constrained layouts to prevent layout shifts

## Typography Hierarchy Context

Subtitle2 fits within the broader Fluent UI typography system as a secondary subtitle component:

- **Display**: Largest text for hero sections and major focal points
- **LargeTitle**: Large headings for page titles and primary focal points
- **Title1, Title2, Title3**: Hierarchical title sizes for main content organization
- **Subtitle1**: Primary subtitle for section organization
- **Subtitle2**: Secondary subtitle for sub-section organization (this component)
- **Subtitle2Stronger**: Emphasized version of secondary subtitle for additional prominence
- **Body1, Body2**: Body text variations for content and descriptions
- **Caption1, Caption2**: Small text for captions, metadata, and auxiliary information

Use Subtitle2 when you need secondary subtitle-level prominence that creates clear sub-organization within content sections, helping users navigate complex information hierarchies while maintaining the overall design system consistency.
