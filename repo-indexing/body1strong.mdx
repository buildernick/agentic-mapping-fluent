# Body1Strong Components

## Overview

### Purpose

Use this component for strong body text styling

### When to Use

The Body1Strong component is designed for emphasized body text content that requires more visual weight than standard Body1 text. Use Body1Strong when you need:

- Text that requires emphasis or attention within body content
- Important statements, highlights, or key information in articles or descriptions
- Enhanced readability for critical content without changing font size
- Consistent typography that follows the Fluent Design System's text hierarchy with stronger visual weight
- Introductory paragraphs, lead text, or callout content that needs to stand out
- Form field labels or descriptive text that requires emphasis
- Text content that serves as a visual bridge between regular body text and headings

### Architecture

**No Coupling** - Body1Strong is a standalone typography component that doesn't require other components to function. It's part of the broader typography system but operates independently, providing emphasized body text styling.

### Component Interdependencies

Body1Strong has no required dependencies and can be used as a standalone component. However, it's part of a cohesive typography hierarchy that includes related variants for different emphasis levels and font sizes.

## Components

### Body1Strong {Standalone Component}

**Purpose:** Provides emphasized body text styling that adheres to the Fluent Design System's typography standards. This component ensures consistent font sizing, font weight, line height, and spacing for readable emphasized body content while maintaining the same font size as regular Body1.

**Interface:**

```typescript
interface Body1StrongProps {
  /**
   * The content to be rendered as emphasized body text
   */
  children?: React.ReactNode;

  /**
   * The HTML element or React component to render
   * @default 'span'
   */
  as?: React.ElementType;

  /**
   * Applies block-level styling (display: block)
   * @default false
   */
  block?: boolean;

  /**
   * Prevents text wrapping (white-space: nowrap)
   * @default false
   */
  nowrap?: boolean;

  /**
   * Applies text truncation with ellipsis on overflow
   * @default false
   */
  truncate?: boolean;

  /**
   * Text alignment
   */
  align?: "start" | "center" | "end" | "justify";

  /**
   * Additional CSS class names
   */
  className?: string;

  /**
   * Inline styles
   */
  style?: React.CSSProperties;
}
```

**Usage Example:**

```tsx
import { Body1Strong } from "@fluentui/react-text";

function Example() {
  return (
    <Body1Strong>
      This is emphasized body text with strong styling that draws attention
      while maintaining readability and consistency with the design system.
    </Body1Strong>
  );
}
```

## Component Relationships

### Typography Hierarchy

Body1Strong is part of a comprehensive typography system that includes:

- **Display**: Largest text for major headings
- **LargeTitle**: Large titles for section headers
- **Title1, Title2, Title3**: Hierarchical title sizes
- **Subtitle1, Subtitle2**: Subtitle variations
- **Body1**: Standard body text
- **Body1Strong**: Emphasized body text (this component)
- **Body1Stronger**: Strongly emphasized body text
- **Body2**: Smaller body text for secondary content
- **Caption1, Caption2**: Small text for captions and labels

### Related Components

```tsx
      </Body1Strong>

        This is the main body text using Body1 for standard paragraph content.
        It provides optimal readability for longer text passages.
      </Body1>

        This paragraph uses Body1Strong again to emphasize important points

        This paragraph uses Body1Stronger for even stronger emphasis when

      </Body1Stronger>

      <Body2 as="p">
        This is supporting text using Body2, which is smaller and suitable for
        secondary information.
      </Body2>
    </article>
  );
}
```

## Best Practices

### Usage Guidelines

- Use Body1Strong for content that needs emphasis while maintaining the same size as Body1
- Ideal for introductory paragraphs, key statements, or important information that should stand out
- Combine with semantic HTML elements using the `as` prop for proper accessibility and meaning
- Use sparingly to maintain visual hierarchy - too much emphasized text reduces its effectiveness
- Perfect for form labels, section introductions, and highlighted content within longer text passages

### Accessibility

- Use semantic HTML elements (`as="strong"` for emphasis, `as="p"` for paragraphs)
- Ensure emphasized text has clear meaning and purpose beyond just visual styling
- Test with screen readers to ensure emphasis is properly conveyed to assistive technologies
- Avoid overusing emphasized text as it can become overwhelming for users with reading difficulties
- Maintain sufficient color contrast when customizing text colors

### Performance

- Body1Strong components are lightweight and have minimal rendering overhead
- Use consistent typography components rather than custom CSS to maintain design system coherence
- Avoid excessive nesting of text components within each other
- The emphasized styling is optimized for performance across different devices and browsers

### Design Consistency

- Stick to the typography hierarchy provided by the design system
- Use Body1Strong for moderate emphasis and Body1Stronger for stronger emphasis
- Maintain consistent spacing and alignment with other typography components
- Use the `block` prop for paragraph-style content to ensure proper line spacing
