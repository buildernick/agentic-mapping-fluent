# Caption2 Components

## Overview

### Purpose

Use this component for secondary caption text styling

### When to Use

The Caption2 component is designed for minimal, secondary caption-level text content that provides the smallest readable font size in the Fluent Design System typography hierarchy. Use Caption2 when you need:

- Secondary metadata, timestamps, or auxiliary information that doesn't compete with primary content
- Minimal supporting text that provides context without drawing attention away from main content
- Small text for footnotes, disclaimers, or supplementary information that users may reference but isn't critical
- Consistent typography for the smallest text content that maintains readability while preserving visual hierarchy
- Helper text, tips, or guidance that supports primary content without being prominent
- Small status indicators, version numbers, or copyright text that needs to be present but unobtrusive
- Label text for less important form fields, data points, or interface elements that provide secondary context
- Fine print, legal text, or terms that must be displayed but shouldn't dominate the interface

### Architecture

**No Coupling** - Caption2 is a standalone typography component that doesn't require other components to function. It's part of the broader typography system but operates independently, providing the smallest caption-level text styling available in the Fluent Design System hierarchy.

### Component Interdependencies

Caption2 has no required dependencies and can be used as a standalone component. However, it's part of a cohesive typography hierarchy that includes related variants for different emphasis levels and font sizes, representing the smallest text size in the caption family.

## Components

### Caption2 {Standalone Component}

**Purpose:** Provides the smallest caption text styling that adheres to the Fluent Design System's typography standards. This component ensures consistent minimal font sizing, appropriate font weight, line height, and spacing for secondary caption-level content while maintaining excellent readability at the smallest recommended text size.

**Interface:**

```typescript
interface Caption2Props {
  /**
   * The content to be rendered as secondary caption text
   */
  children?: React.ReactNode;

  /**
   * The HTML element or React component to render
   * @default 'span'
   */
  as?: React.ElementType;

  /**
   * Applies block-level styling (display: block)
   * @default false
   */
  block?: boolean;

  /**
   * Prevents text wrapping (white-space: nowrap)
   * @default false
   */
  nowrap?: boolean;

  /**
   * Applies text truncation with ellipsis on overflow
   * @default false
   */
  truncate?: boolean;

  /**
   * Text alignment
   */
  align?: "start" | "center" | "end" | "justify";

  /**
   * Applies the italic font style to the content
   * @default false
   */
  italic?: boolean;

  /**
   * Applies the strikethrough text decoration to the content
   * @default false
   */
  strikethrough?: boolean;

  /**
   * Applies the underline text decoration to the content
   * @default false
   */
  underline?: boolean;

  /**
   * Wraps the text content on white spaces
   * @default true
   */
  wrap?: boolean;

  /**
   * Additional CSS class names
   */
  className?: string;

  /**
   * Inline styles
   */
  style?: React.CSSProperties;
}
```

**Usage Example:**

```tsx
import { Caption2 } from "@fluentui/react-components";

function Example() {
  return <Caption2>Last updated 2 minutes ago</Caption2>;
}
```

## Component Relationships

### Typography Hierarchy

Caption2 is part of a comprehensive typography system that provides the smallest text size in the Fluent Design System hierarchy:

- **Display**: Largest text for major headings
- **LargeTitle**: Large titles for section headers
- **Title1, Title2, Title3**: Hierarchical title sizes
- **Subtitle1, Subtitle2**: Subtitle variations
- **Body1, Body1Strong, Body1Stronger**: Body text variants
- **Body2**: Smaller body text for secondary content
- **Caption1, Caption1Strong, Caption1Stronger**: Standard and emphasized caption text
- **Caption2**: Smallest caption text for minimal information (this component)

## Best Practices

### Usage Guidelines

- Use Caption2 for the smallest text content that still needs to be readable and accessible
- Perfect for timestamps, version numbers, copyright text, or other supplementary information
- Combine with semantic HTML elements using the `as` prop for proper accessibility and meaning
- Avoid using Caption2 for critical information - use larger text components for important content
- Ideal for footer text, disclaimers, helper text, or metadata that provides context without competing for attention

### Accessibility

- Ensure sufficient color contrast when using Caption2 due to its small size
- Use semantic HTML elements (time, small, etc.) with the `as` prop for screen reader compatibility
- Consider that very small text may be difficult for users with visual impairments
- Provide alternative ways to access information presented in Caption2 for accessibility compliance

### Performance

- Caption2 renders efficiently as it's a lightweight wrapper around the base Text component
- Use Caption2 consistently throughout your application to maintain typography performance
- The component inherits all performance optimizations from the underlying FluentUI Text component
