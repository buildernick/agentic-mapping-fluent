# Title2 Component

## Overview

### Purpose

Use this component for secondary title text styling

### When to Use

- When you need to display a second-level heading in your interface hierarchy
- For section titles that are subordinate to a primary heading (Title1)
- When creating visual hierarchy in content with multiple heading levels
- In forms or dialogs where you need to introduce secondary sections
- For subsection headers within complex layouts or dashboards

### Architecture

**No Coupling** - This is a standalone component that can be used independently without requiring other components or shared context.

Title2 is a text styling component that provides consistent secondary title typography according to FluentUI design system guidelines. It's part of the typography component family but operates independently.

## Components

### Title2

**Purpose:** Provides styled text rendering for secondary level titles with consistent typography, spacing, and theming support.

**Interface:**

```typescript
interface Title2Props extends React.HTMLAttributes<HTMLHeadingElement> {
  /**
   * Content to be displayed as the secondary title
   */
  children?: React.ReactNode;

  /**
   * Custom CSS class name to apply to the title element
   */
  className?: string;

  /**
   * Inline styles to apply to the component
   */
  style?: React.CSSProperties;

  /**
   * Whether the text should not wrap
   * @default false
   */
  nowrap?: boolean;

  /**
   * Whether to render as a block element
   * @default false
   */
  block?: boolean;

  /**
   * Custom HTML tag to render
   * @default "h2"
   */
  as?: keyof JSX.IntrinsicElements;
}
```

**Usage Example:**

```tsx
import { Title2 } from "@fluentui/react-components";

function Example() {
  return (
    <div>
      <Title2>Section Overview</Title2>
      <p>This is the content under the secondary title.</p>

      <Title2 className="custom-title">Custom Styled Title</Title2>

      <Title2 nowrap>This title will not wrap to multiple lines</Title2>

      <Title2 as="h3">Rendered as h3 element</Title2>
    </div>
  );
}
```

## Best Practices

### Usage Guidelines

- Use Title2 for second-level headings that follow Title1 in your content hierarchy
- Maintain consistent spacing between Title2 and surrounding content
- Combine with other typography components (Body1, Caption1) to create clear content structure
- Use semantic HTML heading levels (h2, h3) when appropriate for accessibility

### Accessibility

- The component renders as an h2 element by default, providing proper semantic structure for screen readers
- When using custom `as` prop, ensure the heading level makes sense in your document structure
- Avoid skipping heading levels (h1 â†’ h3) as this can confuse assistive technologies
- Use descriptive text that clearly indicates the content that follows

### Performance

- Title2 is a lightweight component with minimal rendering overhead
- Text content is efficiently rendered without unnecessary re-renders
- Component supports tree-shaking when bundled with modern build tools

### Visual Consistency

- Title2 automatically applies theme-appropriate typography styles
- Respects dark/light mode preferences when used with FluentUI theme provider
- Maintains consistent font size, weight, and spacing across your application
